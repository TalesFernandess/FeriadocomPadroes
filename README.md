# FeriadocomPadroes

Este projeto simula um sistema de montagem de computadores personalizados, inspirado em sites onde o usuário pode escolher peça por peça (CPU, GPU, RAM, etc.) e adicionar opcionais ao seu PC.
A ideia foi construir um sistema flexível e fácil de manter, permitindo adicionar novos tipos de componentes, fornecedores e funcionalidades sem precisar alterar o código já existente.
Além disso, o sistema também demonstra integração com serviços externos, aplicação de validações (como limite de orçamento) e simplificação do uso por meio de uma interface mais direta para o usuário.
