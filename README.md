# FeriadocomPadroes

Este projeto simula um sistema de montagem de computadores personalizados, onde o usuário pode selecionar componentes (CPU, GPU, RAM, etc.) e adicionar opcionais ao seu sistema.
O sistema foi projetado para ser flexível, extensível e de baixo acoplamento, permitindo a inclusão de novos tipos de componentes, fornecedores e funcionalidades sem a necessidade de modificar o código existente.
Além disso, o sistema também integra serviços externos, aplica validações e simplifica o uso por meio de uma interface de alto nível.
